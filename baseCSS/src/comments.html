Comentários sobre o Base CSS Framework

1) Qual o grau de dificuldade de utilização do mesmo? Por quê?

O framework em si é simples de se utilizar pois não possui nem tantas classes assim para ficar confuso. Grande parte das classes são para organização de layout como content, row e col-X, para divisão em grid semelhante ao Bootstrap. Ele pode ser um pouco complicado de instalar para quem nunca mexeu com NodeJS ou Git, mas quem já conhece é tranquilo.

2) O framework necessita de algum elemento adicional em sua instalação para funcionar?

Sim, por utilizar o SCSS o framework precisa do NodeJS rodando para compilar os arquivos.

3) Quais os pontos fortes apresentados pelo framework utilizado?

Por utilizar o NodeJS para rodar o site quer dizer que a qualquer alteração do código ele já executa na hora e pode ser visto o resultado rapidamente, agilizando bastante o desenvolvimento.

4) Quais os pontos fracos apresentados pelo framework utilizado?

Poucas classes para utilização, a classe row possui um padding negativo para a esquerda o que atrapalha um pouco na organização dos elementos e a documentação podia ser um pouco melhor também.
