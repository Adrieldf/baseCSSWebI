Instruções para utilizar o Base CSS Framework

1) Acesse o link para baixar o projeto base Aqui

2) Clone o repositório para a sua pasta desejada com o seguinte comando 

git clone git@github.com:getbase/starter.git --depth=1 your-website \
            && cd your-website && rm -rf .git

3) Instale o NodeJS LTS (a v12 funciona, a v15 deu problemas para inicializar) Aqui

4) Dentro da pasta do projeto execute o comando para instalar os modulos do node e após o comando start para inicializar o site 

npm i && npm start

5) Se não ocorreu nenhum erro o site deve estar rodando e uma página em branco deve estar aberta

6) Para saber quais classes utilizar acesse o link Classes CSS
